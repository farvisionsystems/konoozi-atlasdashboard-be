# Atlas

## Project setup
- Install Elixir/OTP and Erlang versions from `.tool-versions` (asdf recommended).
- Install npm dependencies for assets: `npm install`.
- Fetch and compile backend deps: `mix deps.get` and `mix compile`.
- Setup database and seeds: `mix ecto.setup`.
- Build assets for dev: `npm run dev` (or `npm run build` for prod).
- Start Phoenix: `mix phx.server` or `iex -S mix phx.server`.
- Visit `http://localhost:4000`.

Ready to run in production? Please [check our deployment guides](https://hexdocs.pm/phoenix/deployment.html).
